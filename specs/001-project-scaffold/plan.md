**Input**: Feature specification from `/specs/001-project-scaffold/spec.md`

## Summary

Establish a reproducible Python project scaffold for `bookmark-mgr`: `pyproject.toml` with Click ≥ 8.0 as a runtime dependency and pytest ≥ 8.0 + aiosqlite ≥ 0.20 as dev-only dependencies; `src/bookmarkcli/` src-layout with a Click entry point; `tests/` discovery directory; `conftest.py` exit-code normalisation; and a locked `uv.lock`. The scaffold must allow `uv sync && bookmarkcli --help && pytest -q` to all exit 0 on a fresh clone.
