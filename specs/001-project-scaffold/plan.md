## Summary

Establish a reproducible Python project scaffold for `bookmarkcli`: a `pyproject.toml`-based package using the `src/` layout, `uv` for dependency management, `Click ≥ 8.0` for the CLI entry point, `pytest ≥ 8.0` as the test runner, and `aiosqlite` as a dev dependency. A root-level `conftest.py` normalizes pytest's exit code 5 ("no tests collected") to 0, satisfying the CI requirement. All decisions were pre-resolved — no clarifications were needed.
